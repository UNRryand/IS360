All about the <a> tag.

Abstract

the <a> tag expresses a hyperlink, which is used to link one page to another page. You can link the pages with absolute referencing, or relative referencing. Absolute referencing uses a location on the web to point the anchor tag towards.
A relative reference points to a local reference of the link on the web server.To use the anchor tag properly, you will need a few attributes.

Details

Here are the attributes you will need to properly use the anchor tag.There are more attributes that are beyond the scope, but the main one you need is:

  • href

Examples

To use the anchor tag properly for an absolute reference, use this as a guide for linking between pages:
<a href="https://www.unr.edu/"></a>
 This is a link to the unr website. You need to have the URL in the href attribute and some text to describe the link in the alt category. Using a relative link is similar, just point to a page that is in your directory.
<a href="./index.html"></a>
 You can also wrap text or a picture in the anchor tag.
<a href="./index.html">wrap</a>
 <a href="https://www.unr.edu/Images/president/im/basic-logos/downloads/nevada-blockn-blue.png"><img src="https://www.website.com/png" alt="text about the picture." /></a>
